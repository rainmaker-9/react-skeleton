## Getting Started

1. **Install dependencies:**

   **Linux/macOS:**

```bash
npm install
```

**Windows (Command Prompt):**

```cmd
npm install
```

2. **Start the development server:**

   **Linux/macOS:**

```bash
npm run dev
```

**Windows (Command Prompt):**

```cmd
npm run dev
```

3. **Build for production:**

   **Linux/macOS:**

```bash
npm run build
```

**Windows (Command Prompt):**

```cmd
npm run build
```
